# typescript-hello-world

Rohas project initialized with typescript handlers.

## Getting Started

1. Generate code:
   ```bash
   rohas codegen
   ```

2. Start development server:
   ```bash
   rohas dev
   ```

   Or start with workbench UI:
   ```bash
   rohas dev --workbench
   ```

3. Validate schema:
   ```bash
   rohas validate
   ```

## Project Structure

- `schema/` - Schema definitions (.ro files)
- `src/handlers/` - Your handler implementations
- `config/` - Configuration files
