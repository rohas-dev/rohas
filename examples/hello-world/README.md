# hello-world

Rohas project initialized with python handlers.

## Getting Started

1. Generate code:
   ```bash
   rohas codegen
   ```

2. Start development server:
   ```bash
   rohas dev
   ```

3. Validate schema:
   ```bash
   rohas validate
   ```

## Project Structure

- `schema/` - Schema definitions (.roh files)
- `src/handlers/` - Your handler implementations
- `config/` - Configuration files
